# Windows 系统禁止更新

# 方式一

# 步骤一：打开命令提示符

1. 在系统搜索框内输入 “CMD”。
2. 右键点击 “命令提示符”，选择 “以管理员身份运行”。（按下 Win+R 键，打开运行对话框。）
3. 输入 “CMD” 并点击确定。

步骤三：设置暂停更新的时间

1. 回到 Windows 更新界面。

2. 点击 “暂停更新” 选项，此时你会发现暂停更新的上限不再是一个月，而是可以设置到最长 100 年。
