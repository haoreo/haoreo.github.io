# 关闭自动系统更新

# 一、进入工厂模式并设置：

1. 按遥控器上的设置键 ，移动光标至 更多设置并按 OK 键确认。
2. 依次进入声音 >> 高级设置 。
3. 把光标移至声音平衡 0 处，OK 键和菜单键，连续交替按三次（电视将自动进入工厂模式）。
4. 接着在工厂模式下，按菜单键后，依次进入 工厂模式 菜单 >> 选项 >> to fac 。
5. 右选项改 U 为 M 后，按遥控器上的关机 / 待机键关机。

# 二、调出原始设置页面，打开开发者模式

1. 按遥控器重新开启电视机，接着按设置键 ，移动光标至更多设置并按 OK 键确认 。
2. 把光标移至关于处。连续按菜单键 4 次（电视将自动进入系统设置页面） 。
3. 在该页面依次进入系统 >> 关于手机，接着把光标移至最后版本号处。然后连续按 OK 键 6 次以上，会提示 已打开开发者模式 。
6. 接着按返回键返回上一级，进入开发者模式选项，找到自动系统更新将它关闭。

打开应用设置 >> 数据管理 >> 配置接口源，可以手机扫码远端导入
