# 外星人本地恢复原厂 SupportAssist OS Recovery 系统

# 一、准备

  1. 16G U 盘一个
  2. 下载 Dell OS Recovery Tool：Dell-USB-Recovery-Tool-Application_VYFNW_WIN_2.3.7012.0_A00

# 二、开始制作启动盘

# 1 安装 Dell OS Recovery Tool

# 2 插上 U 盘，并打开 Dell OS Recovery Tool

# 3 选择高级恢复 > 从可用的操作系统映像中选择 > 对于此计算机。（自动识别当前机型序列号）

# 4 选择刚才插的 U 盘，U 盘中数据提前做好备份，因为会格式化

# 5 下载中

# 6 制作启动盘完成

# 三、开始重新安装

# 1 重启一直敲击 F12

# 2supportassist 助手启动中

# 3 进入助手后联网，尽量插网线，选右下角重置功能

# 4 进入选择重置界面，本地重置为出厂设置

# 5 这里注意，确保电脑中有没有要备份的数据做好备份，没有然后点下一步

# 6 同意协议下一步

# 恢复中，恢复出厂预装版

# 8 恢复完成，重启

# 5 恢复完成一顿操作，哎呀，一言不合就又恢复到买来时的样子了
