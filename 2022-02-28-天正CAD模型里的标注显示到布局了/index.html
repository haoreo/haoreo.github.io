# CAD 在布局视口中替换模型空间和图纸空间

在模型里面做的标注，保存后再次打开时，标注却去了布局里面了？不要慌！

# 替换下模型空间和图纸空间

# 执行以下步骤之一：

使用命令：

切换到图纸空间。

切换到模型空间。

显示 “模型 / 图纸” 切换按钮

  1. 在 AutoCAD 命令行中输入
  1. 在 “选项” 对话框中选择 “显示” 选项卡。
  2. 在 “布局元素” 下，选中 “显示布局和模型选项卡” 复选框。
  3. 单击 “确定” 。
  4. “模型/图纸” 切换按钮将显示在状态栏上。

选择 “模型” 以激活模型空间。选择 “PAPER” 以激活 “Paper Space” 。

双击切换

  • 在视口内双击以激活模型空间。
  • 在视口外双击以激活图纸空间。
