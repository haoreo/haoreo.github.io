# Mac 安装打开错误异常解决方法

# 安装软件，无法打开提示来自身份不明开发者

在 MAC 下安装一些软件时提示 "来自身份不明开发者"

其实这是 MAC 新系统启用了新的安全机制。默认只信任 Mac App Store 下载的软件和拥有开发者 ID 签名的应用程序。 换句话说就是 MAC 系统默认只能安装靠谱渠道（有苹果审核的 Mac App Store）下载的软件或被认可的人开发的软件。

这当然是为了用户不会稀里糊涂安装流氓软件中招，但没有开发者签名的 “老实软件” 也受影响了，安装就会弹出下图所示警告框：

“打不开 xxx，因为它来自身份不明的开发者”。

# 出现这个问题的解决方法有 2 种：

  • NO.1 最简单的方式：按住 Control 后，再次点击软件图标，即可。

  • NO.2 修改系统配置：

    1. 系统偏好设置
    2. 安全性与隐私

  1. 修改为任何来源

# 打开显示 “已损坏，无法打开。 您应该将它移到废纸篓”

# 解决方法一

  1. 打开 “访达”（Finder）进入 “应用程序” 目录，找到该软件图标，将图标拖到刚才的终端窗口里面，会得到如下组合 (如图所示)：

sudo xattr -r -d com.apple.quarantine /Applications/WebStrom.app

  1. 终端窗口按回车，输入密码回车即可。

  2. 接着重新打开安装软件，就可以正常安装了。

# 解决方法二

  1. 打开终端，输入以下命令：（cr 后面输入一个空格）
  1. 将出现已损坏的应用拖入终端中（xattr -cr 的后面），代码样式为

xattr -cr /Applications/imFile.app

  1. 再回车就可以正常打开了
