# 苹果 Mac 恢复原厂系统 Mac OS X

# 一、开始恢复操作

# 1. 将 Macbook 笔记本关机

# 二、恢复选项

# 2. 根据所需选择以下几项之一操作

  • 开机时，同时按住 “Command+R” 或长按开关机键，直至出现恢复选项 / OS X 实用工具（恢复出厂并安装至最新系统）

  • 开机时，同时按住 “Command+Option+Selft+R”，直至出现恢复选项 / OS X 实用工具（恢复出厂并选择安装最近可支持的系统）

# 3. 在实用工具界面。点击磁盘工具。抹掉内置磁盘中的数据

# 4. 抹掉后返回，然后点击 "重新安装 macOS"

# 5. 等待安装完成。

如有在操作过程中有问题，请及时反馈！
