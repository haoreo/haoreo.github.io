# Mac 安装 Homebrew，Git 和 node.js

# 安装 Homebrew

# 1. 在命令行工具输入以下命令之一，安装 Homebrew。

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  • 国外源：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  • 国内源：

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

选择源：推荐 1

# 2.Homebrew 安装 node.js

brew install node

安装后可以使用命令来检查是否安装成功？检查 node

node -v

返回版本结果如：

v12.14.1

检查 npm 是否安装成功，npm 是 node.js 的包管理工具

nmp -v

返回版本结果如：

6.13.4

# 3.Homebrew 安装 git

brew install git

检查 git 是否安装成功

git -v

返回版本结果如：
git version 2.24.3 (Apple Git-128)

# 4. 使用 npm 安装 Hexo

执行以下命令

sudo npm install -g hexo-cli

创建 blog 文件夹

cd ~ && mkdir blog && cd blog

在该目录下执行初始化（下载一些 node.js 的依赖文件）

hexo init

在该目录下执行预览

hexo s

当看到如下输出就可以预览我们创建的博客了

INFO  Validating config INFO  Start processing INFO  Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

预览效果如下

# 5. 注册登录 Github

  • 注册登录 Github
  • 创建一个名字为 username.githug.io 的仓库，选择 Public

# 6. 配置关联 github

在命令行输入配置用户名和账号的命令（如果已经配置过，该步骤可以忽略）

git config --global user.name "jack.github.io"git config --global user.email "jsdk@gmail.com"

通过终端命令生成 SSH Key

ssh-keygen -t rsa -C "email@gmail.com"

执行完成后会在～/.ssh/id_rsa.pub 目录下生成 需要使用的 key。

  • 可以使用命令行输出 key 并复制

cat ~/.ssh/id_rsa.pub

  • 也可以找到这个文件打开它并复制

$open ~/.ssh Users/youname/.ssh/id_rsa.pub

无法找到key文件，则执行依次以下命令

NO.1 ssh-keygen -t rsa
NO.2 cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
NO.3 chmod og-wx ~/.ssh/authorized_keys

再打开文件夹找到文件并复制里面的内容，然后登录你的 github 账号 找到 setting

先点击 SSH and GPG keys 然后再点击 New SSH key 进入到配置 SSH Key 的页面

然后输入复制好的 key 的内容
￼

点击 Add SSH Key ，然后命令行输入

cd blog

执行命令
sudo npm install hexo-deployer-git --save

然后开始修改配置文件
~/Desktop/blog/_config.yml

修改 deploy 部分为

deploy: type: git repo: git@github.com:Johnson8888/Johnson8888.github.io.git branch: master
